This section attempts to describe how to use the com.io7m.jcamera package. The example code uses JOGL but the package can obviously be used under any Java input/windowing system. This section doesn't attempt to explain why anything works the way it does - readers are encouraged to read the design and implementation section, which describes everything in extensive detail.
